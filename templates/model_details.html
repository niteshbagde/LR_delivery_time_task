Train / Re-Train Model 2

This is the model 2 details :

The data feed to model contains columns :

'Delivery_person_Age', 'Delivery_person_Ratings', 'Weather_conditions', 'Road_traffic_density',

'Vehicle_condition', 'Type_of_order', 'Type_of_vehicle', 'multiple_deliveries', 'Festival', 'City',

'Time_taken (min)', 'Daytime', 'order_pick_Time_Difference_Minutes', 'onground_dist(KM)'

Note: The custom columns were added and also removed other columns to increase the accuracy and remove over fitting

  • 'order_pick_Time_Difference_Minutes' - Time difference between ordered time and picked order time by delivery agent
  • 'onground_dist(KM)' - scrapped the realworld distance using google maps (for 1000 points only)
  • 'Daytime' - the time when order was picked by agent
